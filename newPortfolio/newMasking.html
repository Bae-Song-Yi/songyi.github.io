Description
신분증 인증 시 마스킹 하기 위한 도구
Project Background
  • ① 마스킹 실수가 많아 현업 업무량 증가
  • ② OCR 도입 고려
마스킹 이벤트
  • ① mousedown, mousemove, mouseup 및 touchstart, touchmove, touchend 이벤트를 사용하여 움직인 거리를 계산해 마스킹 할 수 있도록 해주었습니다.
  • ② 반응형을 고려하여 마스킹의 사이즈 단위는 픽셀 단위 대신 %로 계산해주었습니다.
  • ③ html2canvas 라이브러리를 사용하여 이미지 영역을 캡처해 이미지 파일로 변환후 서버에 업로드하는 작업을 수행 했습니다.
JPG, JPEG, PNG, PDF 속성 파일 첨부 가능
(카메라 가로모드로, 서류가 가득차게 촬영해주세요)
신분증 사본 제출하기

이름, 생년월일, 사진, 발급일을 제외한 모든 정보를 마스킹합니다.

  • 1. 파일은 JPG, JPEG, PNG 만 선택 가능합니다.
  • 2. 파일 선택 시 미리보기가 가능하고, 드래그하여 이미지 위에 마스킹 영역을 지정할 수 있습니다. (팬 모양 아이콘을 눌러 비활성화 가능)
  • 3. 만약, 실수했다면 초기화하여 마스킹을 모두 지울 수 있습니다.
  • 4. 모바일 마스킹 시 핀치줌(두손가락 터치 후 벌리기)으로 확대하여 이용하시면 편리합니다.
  • 파일선택
  • 예시 파일선택

Project Result
  • ① OCR 프로그램 도입하려 했으나, 비용 절감과 작업의 단순화를 위해 자체개발 했습니다.

    - OCR 도입 시 건당 금액 : 약 1달러

  • ② 자체 개발을 통해 현업 업무량 65% 감소 했습니다.
