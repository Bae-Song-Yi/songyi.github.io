Description
GPT API를 활용하여 이미지의 텍스트 추출
Project Background
  • ① 회원가입 시 입력사항이 많아 불편함을 줄이기 위해 GPT API를 활용하여 이미지 내 텍스트를 자동으로 추출하려고 함
POSTMAN을 통해 프롬프트 적용된 GPT 답변 취득
  • ① POSTMAN 플랫폼을 통해 GPT API에 이미지 내용을 최대한 정확하게 추출하기 위해 여러 번 질문을 하여 정확도를 90%까지 높였습니다.
  • ② GPT API에 POST 요청하여, 이미지의 텍스트 내용을 분석하고 이를 JSON 형식으로 반환시켜 주었습니다.
  • ③ escapeHtml 함수를 사용해 HTML로 변환하여 페이지에 표시해 줄 수 있도록 구현했습니다.
  • ④ API 호출에 실패할 경우, 오류 메시지를 페이지에 표시하도록 구현해 구현했습니다.
