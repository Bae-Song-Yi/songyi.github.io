Description

Period

Skill

Front
100%
Project Background
  • PC 환경에 최적화 되어있는 서비스
    → 모든 디바이스 최적화 필요
  • 응찰 버튼 실수로 클릭하는 상황
    → 응찰 버튼 민감도 강화
Main features
  • 스켈레톤 UI 적용 (데이터를 가져오는 동안 보여주는 화면)
  • 응찰가 변경됐을 경우 시각적 효과를 주기위한 애니메이션 기능 적용
    -> 금액 요소 깜박이 애니메이션
  • 진행중인 LOT 클릭시 현재 진행중인 작품 자동 포커싱
  • 모바일의 경우 응찰버튼 민감도를 강화하기 위해 잠금 기능 적용
