Code Review3

리엑트 변환 작업 하고있어요!

  • 1. 텍스트 박스 색상은 작품에 따라 달라지므로 동적으로 처리해주었고, 개발시 확인하기 편하도록 인라인 스타일로 적용했습니다.
  • 2. 카드 리스트 형식의 값들은 useState를 사용하여 샘플 데이터값들을 넣어주었고, 초기 값을 설정하여 노출시켜주었다.
    무한 스크롤 기능은 부트스트랩 없이 직접 개발 해주었습니다.
    useCallback을 사용하여 handleScroll 함수가 상태 값 변화에 따라 효율적으로 재생성되도록 하였으며, 불필요한 함수 재생성을 방지하여 성능을 최적화 해줄 수 있도록 처리하였습니다.
  • 3. useRef 훅과 같이 컴포넌트를 처음 렌더링할 때 null 또는 undefined인 경우 에러가 발생하지 않도록, 옵셔널 체이닝 연산자를 사용하여 undefined를 반환하도록 처리했습니다.
  • 4. 탭영역 같이 반복되는 컴포넌트를 렌더링 할 떄 map()내장함수를 사용하여 배열에 문자열을 입력 해준 후 a태그를 반환하도록 처리했습니다.
